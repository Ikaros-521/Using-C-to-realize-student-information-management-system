# Using-C-to-realize-student-information-management-system
学生信息管理系统

# 使用说明
终端执行 make  编译

终端执行 ./sims.bin 运行程序

# 其他
超级管理员账号：admin 
密码：admin

用户账号和密码，学生账号和密码以二进制形式存储在data目录下的2个文件中

头文件存储在 include 文件夹下

#### 相关问题
1.如果异常退出程序，请删除data中的数据文件重新创建
