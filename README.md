
# 学生信息管理系统C

#### 介绍
学生信息管理系统，开发环境：Ubuntu12.04 LTS

#### 使用说明

1. 终端输入 make
2. 终端输入 ./sims.bin
3. 管理员账号和密码都是admin
4. 用户的账号我这里是 账号 a 密码 1
5. 用户账号和密码，学生账号和密码以二进制形式存储在data目录下的2个文件中
6. 头文件存储在 include 文件夹下
